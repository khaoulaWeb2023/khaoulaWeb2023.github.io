Introduction au monde du développement Web

Le développement web est l'ensemble des activités liées à la conception, la création et la maintenance de sites web et d'applications web, en utilisant une infrastructure backend pour stocker les données et une interface frontend pour permettre aux utilisateurs d'interagir avec le système.

Cest quoi un framework ?

Essentiellement, un framework est une boîte à outils pour un développeur web. Littéralement, « frame » veut dire « cadre » tandis que « work » signifie « travail ». Un framework contient donc des composants autonomes qui servent à faciliter et à optimiser la création d’un site web ou d’une application. Il existe 2 type de Framework en developpement web :le frontend et backend

Cest quoi un framework Frontend ?

Le frontend : fait référence à la partie d'une application web qui est visible pour l'utilisateur et avec laquelle il peut interagir directement. Cela comprend l'interface utilisateur, la mise en page, les styles, les animations et les fonctionnalités interactives telles que les boutons, les formulaires et les menus déroulants. Le frontend est généralement développé en utilisant des technologies web telles que HTML, CSS et JavaScript, ainsi que des frameworks et des bibliothèques de développement tels que React, Angular, Vue, Bootstrap et jQuery.


Cest quoi un framework Backend ?

Le backend : à la partie invisible d'une application web qui gère les données, les requêtes de l'utilisateur et les fonctionnalités de l'application qui ne sont pas directement visibles par l'utilisateur. Cela comprend le stockage de données, la gestion des utilisateurs, l'authentification, l'autorisation, le traitement des formulaires et la communication avec d'autres services.
